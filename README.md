## Dependencies

>1 Step. Add the code to your  `settings.gradle` file
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

>2 Step. Add the code to your  `build.gradle` file
```
	dependencies {
	        implementation 'com.github.salikhdev:costume-dialog:1.0.0'
	}
```
